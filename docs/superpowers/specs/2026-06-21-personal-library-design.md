## 1. 项目概要

### 1.1 这是什么

一个**公开的个人思维花园网站**。把读书笔记、文字创作、游戏感悟、音乐评价、电影评价编织成一张可视化的知识图谱网。既是个人外置大脑，也是分享品味与思考的公开空间。

### 1.2 用户画像

- **主人（你）**：ENFP，思维跳跃，多线程爱好者。每周几条记录，有感触才写。有少量 Python 和 JS 基础。
- **来访者**：偶然或刻意访问你网址的人，浏览你的阅读品味、思想轨迹。

### 1.3 核心体验

- 🕸️ **侦探白板**：首页是一张可拖拽缩放的交互式知识图谱——节点 = 笔记卡片，连线 = 关联
- ➕ **前台速记**：直接在白板上点击新建，系统自动扫描标签/关键词/作者，建议连线
- 🔍 **多维浏览**：图谱模式 / 列表分类浏览 / 关键词搜索 / 随机漫步
- 🏷️ **灵活分类**：五种内容类型各有定制字段，标签系统贯穿全局

### 1.4 视觉方向：Saul Bass 式手绘图谱风

**为什么是 Saul Bass：**

Saul Bass 的图形语言介于手绘与设计之间——粗粝线条、有机形状、手写感字体。恰好对应「侦探工作室软木板 + 大头针 + 手写标签」的气质。节点不是冰冷的圆点，而像一张张手撕的便签。

**核心视觉约束：**

| 要素 | 规范 |
|------|------|
| 底色 | 暖纸色系（如 `#F5F0E8` 旧纸色），非纯白 |
| 卡片形状 | 圆角矩形，微锯齿纸片边缘感（`border-radius` 不对称 + 微 box-shadow 投影） |
| 节点颜色 | 按类型区分——电影砖红 `#B8404A`、书墨蓝 `#3D5A73`、音乐姜黄 `#C68B3C`、游戏苔绿 `#5C7A56`、原创文字奶油 `#E8D5B7` |
| 连线 | 非直线，带轻微弧度和抖动的"麻绳线"，悬停高亮 + 浮现关联原因 |
| 字体 | 标题用手写/衬线字体（如 ZCOOL KuaiLe 或 思源宋体），正文用可读的衬线体 |
| 图标 | 手绘风格 SVG 内联图标，不用 emoji |
| 装饰 | 大头针 SVG 图钉住卡片、软木板纹理背景（CSS 渐变模拟）、弧线箭头 |

**设计哲学约束（给实现用）：**
- 连线不要几何完美——允许轻微偏移，模拟手画
- 卡片不要精确等大——同类型可以有 5-10px 的随机浮动
- 悬停动画用缓出曲线，模拟纸张被轻轻提起来的感觉
- 空白处放"线索"——比如一句随机展示的金句，"还没有连线等着被发现"

## 4. 功能详述

### 4.1 侦探白板（首页 · 图谱模式）

**交互：**
- 节点按类型着色，拖拽可移动，滚轮缩放，空白区拖拽平移
- 连线自动弯曲，已确认 = 实线，待确认 = 虚线
- 点击节点 → 侧边滑出该卡片完整内容（阅读模式）
- 悬停连线 → 浮现关联原因
- 点击空白区 → 关闭侧边详情

**新建卡片流程：**
1. 点击白板上 [+ 新建卡片] 浮动按钮
2. 弹出表单，先选类型（书/影/音/游戏/创作）→ 表单字段随之切换
3. 填写核心字段（标题必填，其余可选，标签可输入已有标签或直接创建新的）
4. 保存 → 卡片节点出现在白板上
5. 后端扫描已有节点 → 返回匹配到的建议连线（虚线闪烁显示）
6. 你可以悬停虚线 → 确认（变实线）或驳回（消失）

### 4.2 自动连线规则

| 匹配方式 | 权重 | 示例 |
|---------|------|------|
| 相同标签 | 高 | 「存在主义」同时出现在一条书评和一条电影评价 |
| 同一创作者 | 高 | 同一作者/艺术家/导演的不同作品 |
| 标题关键词重叠 | 中 | 标题都提到「时间」 |
| 正文关键词重叠 | 低 | 正文出现相同的人名/书名/概念 |

> 连线建议只在新卡片保存时触发（不全局重算），保证性能。

### 4.3 手动连线

- 在图谱上，从一个节点拖出连线到另一个节点
- 弹出小输入框，填「为什么连着」（可选）
- 保存后生成实线

### 4.4 其他浏览模式

| 模式 | 说明 |
|------|------|
| 📜 列表/卡片墙 | 按时间倒序排列所有卡片，左侧按类型/标签筛选。卡片以纸片样式排列 |
| 🔍 搜索 | 输入关键词，搜索标题 + 正文 + 标签；结果高亮显示 |
| 🎲 随机漫步 | 点击 → 跳转到随机一个节点，底部显示「和你当前浏览过的内容可能有关的几条」 |

### 4.5 后台管理（Django Admin）

- 列表视图：所有卡片、可按类型/时间/标签筛选
- 编辑已有卡片
- 批量添加标签
- 管理连线（查看、手动创建、删除错误连线）

## 5. 技术选型

| 层 | 选型 | 理由 |
|------|------|------|
| 后端框架 | Django 5.x（Python） | 自带 admin 后台，ORM 强大，你有 Python 基础 |
| 数据库 | SQLite | 零配置，免费，对个人网站绰绰有余 |
| 前端图谱 | Cytoscape.js | 专业的图可视化库，动画流畅，可定制 |
| 前端框架 | 原生 JS + Cytoscape.js | 不引入 React/Vue 等重框架；图谱用 Cytoscape，页面交互用原生 JS |
| CSS 方案 | 手写 CSS + CSS 自定义属性（变量） | 完全控制设计细节 |
| 部署 | PythonAnywhere 免费档 | 免费，Django 原生支持 |
| 代码托管 | GitHub | 免费 |

## 6. 部署方案

| 项目 | 方案 | 费用 |
|------|------|------|
| 数据库 | SQLite（文件型，无需单独服务） | 免费 |
| Web 托管 | PythonAnywhere 免费档 | 免费 |
| 代码托管 | GitHub | 免费 |
| 域名 | `你的名字.pythonanywhere.com`（自带） | 免费 |
| 图片存储 | 本地 media 目录（PythonAnywhere 免费档有少量空间限制） | 免费 |

**PythonAnywhere 免费档注意事项：**
- 每 3 个月需登录续期（点按钮即可）
- 无访客时会休眠，首次打开多等 2-3 秒
- 以后想用自己的域名 → 升级到 Hacker 档（约 $5/月）

### 6.1 备选方案：WordPress 方案

如果将来 Django 方案因故不合适，WordPress 备选：

| 项目 | 方案 | 费用 |
|------|------|------|
| CMS | WordPress.org（自托管） | 免费 |
| 主机 | 免费主机（如 InfinityFree） | 免费 |
| 内容类型 | Custom Post Types + Advanced Custom Fields | 免费插件 |
| 图谱 | WP GraphQL + 前端 Cytoscape.js | 免费但需开发 |

WordPress 方案的劣势：免费主机稳定性较差；服务器端逻辑（自动连线）不如 Python 灵活；PHP 你不熟，改起来费劲。因此作为备选，不建议优先。

## 7. 不在本期范围

以下是有意不纳入第一版的：

- 移动端原生 App（网站本身做响应式，手机浏览器可用）

- 多语言支持
- 数据导入/导出工具
- 邮件订阅

## 8. 设计约束 · Saul Bass 风格落地清单

### 8.1 颜色

```
--bg-paper:     #F5F0E8     /* 旧纸底色 */
--bg-cork:      #C4A882     /* 软木板色（背景纹理用） */
--ink-primary:  #2C2416     /* 深棕墨色（主文字） */
--ink-muted:    #8C7B6A     /* 浅棕（辅助文字） */

--card-book:    #3D5A73     /* 书 · 墨蓝 */
--card-music:   #C68B3C     /* 音乐 · 姜黄 */
--card-film:    #B8404A     /* 电影 · 砖红 */
--card-game:    #5C7A56     /* 游戏 · 苔绿 */
--card-writing: #E8D5B7     /* 原创 · 奶油 */

--accent:       #D4844A     /* 点缀色 · 旧铜色 */
--pin:          #CC4444     /* 大头针红 */
```

### 8.2 字体

- 标题：思源宋体（Noto Serif SC）或 ZCOOL KuaiLe
- 正文：思源宋体（Noto Serif SC）
- 代码/标签：霞鹜文楷（LXGW WenKai）

### 8.3 动画

- 节点悬浮：`cubic-bezier(0.34, 1.56, 0.64, 1)` ——轻微的弹性，模拟纸被提起
- 连线出现：400ms 缓出
- 卡片侧滑：300ms
- 不接受生硬的线性动画

### 8.4 不可妥协的原则

- 不用纯白底色
- 不用纯黑文字
- 不用 emoji 做图标（用 SVG 手绘风图标）
- 连线不做完美的几何直线（允许 1-3px 的微弧偏移）
- 卡片不精确等大（同类型有 5-10px 的随机浮动范围）
- 不出现赛博霓虹 / 深蓝底 `#0D1117` 色系
