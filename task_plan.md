# 个人图书馆 · 实现计划

> 状态：规划中 | 设计文档：`docs/superpowers/specs/2026-06-21-personal-library-design.md`

---

## Phase 0：项目脚手架

**目标**：Django 项目初始化、基础配置、本地可运行

- [ ] 安装 Python 3.x + Django 5.x
- [ ] 创建 Django 项目 `personal_library`
- [ ] 配置 SQLite 数据库
- [ ] 配置静态文件路径
- [ ] 创建基础模板结构
- [ ] 本地运行验证（`python manage.py runserver`）
- [ ] 初始化 Git 仓库（已完成）

**产物**：可本地访问的空白 Django 站点

---

## Phase 1：数据模型

**目标**：所有 Model 定义完毕，Django Admin 可录入数据

- [ ] 抽象基类 BaseCard（公共字段：created_at, updated_at, is_published, tags）
- [ ] 创建 5 个具体 Card Model：BookCard, MusicCard, FilmCard, GameCard, WritingCard
- [ ] Connection Model（节点A、节点B、连线原因、连线类型、是否确认）
- [ ] Tag Model
- [ ] 注册所有 Model 到 Django Admin（定制 list_display, list_filter, search_fields）
- [ ] 创建并运行 migrations
- [ ] 在 Admin 中手动录入几条测试数据

**产物**：通过 Admin 可录入和管理所有卡片类型

---

## Phase 2：前台基础页面

**目标**：前台所有页面框架搭建完毕，数据从数据库渲染

- [ ] 定义前台 URL 路由
- [ ] 创建 base.html 模板（共享头部、导航、CSS 变量、字体引入）
- [ ] 首页 — 侦探白板视图（图谱画布容器）
- [ ] 列表/卡片墙视图
- [ ] 卡片详情视图（点击节点后的侧边内容）
- [ ] 搜索视图（含结果高亮）
- [ ] 新建卡片表单视图（弹窗或页面）

**产物**：前台各页面骨架可用，数据从数据库渲染

---

## Phase 3：知识图谱（Cytoscape.js）

**目标**：首页图谱可拖拽缩放，节点和连线正确显示

- [ ] 引入 Cytoscape.js（CDN 或本地）
- [ ] 创建图谱初始化 JS 模块
- [ ] 后端 API：返回所有已发布卡片的 JSON（含节点坐标、类型、连线数据）
- [ ] 前端：解析 API 数据 → 渲染为 Cytoscape 节点 + 连线
- [ ] 节点样式：按类型着色（墨蓝/姜黄/砖红/苔绿/奶油）
- [ ] 连线样式：实线（已确认）/ 虚线（待确认），微弧效果
- [ ] 交互：拖拽节点、滚轮缩放、画布平移
- [ ] 点击节点 → 侧边栏滑出详情
- [ ] 悬停连线 → tooltip 显示连线原因

**产物**：首页图谱完整可交互

---

## Phase 4：新建卡片 + 自动连线

**目标**：前台可新建卡片，系统自动建议连线

- [ ] 前端：新建卡片弹窗表单（类型选择 → 动态字段切换）
- [ ] 后端 API：接收卡片数据 → 保存
- [ ] 自动连线逻辑：扫描标签/作者/关键词 → 生成建议连线
- [ ] 前端：新节点出现 + 建议连线（虚线闪烁）
- [ ] 确认/驳回建议连线
- [ ] 手动连线（拖拽 A → B）

**产物**：前台新建卡片 + 自动连线完整闭环

---

## Phase 5：其他浏览模式

**目标**：列表/卡片墙 + 搜索 + 随机漫步

- [ ] 列表/卡片墙：按时间倒排 + 左侧类型/标签筛选
- [ ] 搜索：关键词搜索标题+正文+标签，结果页高亮
- [ ] 随机漫步：随机跳转节点 + 底部相关推荐

**产物**：4 种浏览模式均可使用

## Phase 6：Saul Bass 视觉落地

**目标**：全部设计约束生效

- [ ] CSS 变量注入：暖纸色系、卡片颜色、点缀色、大头针红
- [ ] 纸张纹理背景（CSS 渐变模拟软木板/纸纹）
- [ ] 卡片纸片效果（微旋转、不对称圆角、投影）
- [ ] 连线麻绳样式（非直线、微抖动）
- [ ] 手绘风格 SVG 图标（书/音乐/电影/游戏/写作/大头针/搜索/随机）
- [ ] 字体引入（思源宋体 + 霞鹜文楷）
- [ ] 动画：弹性悬浮、连线渐变出现、卡片侧滑
- [ ] 空白处随机金句线索

**产物**：网站视觉 100% 匹配 Saul Bass 手绘图谱风

## Phase 7：用户认证

**目标**：前台公开访问，后台仅你登录

- [ ] Django 内置 auth 配置
- [ ] 创建超级用户（你）
- [ ] 前台页面无需登录（公开）
- [ ] 新建卡片接口需登录（防止别人乱写）
- [ ] Admin 需登录

**产物**：公开/私密边界清晰
